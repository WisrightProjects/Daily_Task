Day 3 - HTML Practice Summary

1. Abbreviation Element

Example of abbreviation: HTML

2. Subscript Element

Water formula: H2O

3. Avoid Using <u>

This text is underlined (may look like a hyperlink and confuse users)

4. Open Graph Meta Tags

Open Graph tags are used to control how content appears when shared on social media.

<meta property="og:title" content="My Website"> <meta property="og:description" content="Description here"> <meta property="og:image" content="image.jpg">

5. SEO Title Format

Correct format: Primary Keyword - Secondary Info | Brand Name

Example: Best Laptops - Buying Guide | MyBrand
